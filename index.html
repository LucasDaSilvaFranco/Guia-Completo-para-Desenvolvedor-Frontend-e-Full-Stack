🗓️ Cronograma Semanal (2-3h/dia)

Dia Foco Principal

Quarta JavaScript Básico
Quinta JS ES6+ (Arrays, Funções)
Sexta Git/GitHub + Publicar Projetos

Domingo Revisão ou Projeto Livre
